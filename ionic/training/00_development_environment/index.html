Ionic 開發環境

更新於 2017/09/15 | 分類於 Training

開發環境建置步驟:

Ionic 開發環境與 Angular 雷同，我們需要安裝：

  • Node.js 6 LTS & npm 3.x.x 以上版本
  • Ionic CLI
  • Cordova
  • Visual Studio Code (簡稱：VS Code)

安裝方式請參考參考 Angular 開發環境(Windows 版)？。
最大差別是 Angular 需要安裝 Angular CLI，Ionic 則是安裝 Ionic CLI。

安裝 Ionic CLI & Cordova

利用命令提示字元透過 npm 安裝 Ionic CLI，指令如下：
npm install -g ionic cordova
目前版本： Ionic 為 3.10.3，Cordova 為 7.0.1
