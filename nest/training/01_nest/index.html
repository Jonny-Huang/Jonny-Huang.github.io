Nest 新手上路

更新於 2017/10/23 | 分類於 Training

前言

之前分享了 學會 Angular 後，就只能寫前端程式嗎？：Nest，概略說明了 Nest 與 Angular 的相似度，部分網友似乎希望能夠再說明深入一點，筆者就針對這幾天的研究再分享一些心得，因為 Nest 在架構上與 Angular 相似度極高，所以筆者是站在有 Angular 開發經驗的角度來做說明。
Nest 底層運作是採用了 Node.js 上知名的 package - Express 與 socket.io，在這基礎上提供一個更有效率的開發模式，比較特別的是透過 Nest 可以很方便的轉換成 微服務(Microservices)架構。
開發上我們以官方提供的 Nest CLI 來建立相關檔案，若未安裝可以透過下列指令安裝：
npm install -g @nestjs/cli

建立新專案：new

nest new demo-nest02
cd demo-nest02
npm install
